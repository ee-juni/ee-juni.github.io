1. Transform

rotate

transform: rotate(45deg);

입력한 각도만큼 (평면적 회전)

scale

transform: scale(x, y);

width를 x배, height를 y배 확대(decimal도 가능)

skew

transfrom: skew(10deg, 20deg);

x축, y축 기준으로 비틂

translate

transform: translate(100px, 200px);

x축, y축 방향으로 옮김

prefix 접두사

-webkit/moz/ms/o-transfrom

각 브라우저 버젼/종류에 따라 실행 vary 가능
(각각 Chrome/Safari, Mozilla, IE9.0, Opera에 해당)

2. Transition

property

transition-property: width;

효과 적용 대상 css 속성

duration

transition-duration: 2s;

효과 표현 지속 시간

timing-function

transition-timing-function: linear;

효과의 속도. linear=일정하게.

delay

transition-delay: 1s;

효과 발동 지연. 1s=1초

가상 선택자: hover

hover { width: 300px; }

CSS 내장 클래스. '마우스 올렸을 때' width를 300px로 변경.

Transition 종합

.. { transition: width 2s linear 1s;}.. hover { width: 30px; }

마우스 올리면 1초 뒤 width가 2초 동안, 속도 일정하게 300px로 변화

3. Animation

Animation

animation-name: changeWidth;

animation-duration: 3s;

animation-timing-function: linear;

animation-delay: 1s;

animation-iteration-count: 6;

animation-direction: alternate;

4. 레이아웃 속성

display

{ display: inline/block/inline-block; }
block과 inline 요소의 성격을 바꿀 수 있음.

float

{ float: left/right; }
왼쪽/오른쪽 끝으로부터 정렬.
선택자를 띄워 새로운 레이어층 만듬.

clear

{ clear: both; }
float를 쓴 바로 다음 요소를 float 아래 겹치지 않게 밀어줌

브라우저와 공간 사이 공백 제거

html, body { margin:0; padding:0; }
0으로 초기화해줌. 선택자를 *로 써도 됨 (*=select all).
