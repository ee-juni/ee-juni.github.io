1. 박스 모델

박스 모델 구조

margin, border, padding, content로 구성됨

margin과 padding의 차이

  • margin-left: border 밖 왼쪽 여백
  • padding-left: border 안 왼쪽 여백
    공간이 여백을 포함한 크기로 변경됨 유의

margin과 padding 작성 방법

Ex) div { margin: 100px 0 0 100px; }
top right bottom left 순서로 한 줄에 작성 가능
padding도 동일한 문법

2. Block, Inline 요소

Block 요소와 Inline 요소

  • block 요소 -> 세로로 continue (줄바꿈 있음).
  • inline 요소 -> 가로로 continue.
    width, height, margin, padding, bottom 모두 적용 불가.
  • 속성: HTML 태그가 지닌 추가 정보
  • 속성값: 어떤 역할을 수행할지 구체적인 명령을 진행하는 것

3. 마진 병합 현상

형제지간의 마진 병합

Ex) .box1 { margin-bottom: 150px; }
.box2 { margin-top: 100px; }
margin-bottom과 margin-top 중 숫자가 큰 값으로 적용

모자지간의 마진 병합

Ex) main>article
article { margin-top: 100px; }
 article의 margin이 부모인 main에게도 적용됨

4. 레이아웃 속성

display

{ display: inline/block/inline-block; }
block과 inline 요소의 성격을 바꿀 수 있음.

float

{ float: left/right; }
왼쪽/오른쪽 끝으로부터 정렬.
선택자를 띄워 새로운 레이어층 만듬.

clear

{ clear: both; }
float를 쓴 바로 다음 요소를 float 아래 겹치지 않게 밀어줌

브라우저와 공간 사이 공백 제거

html, body { margin:0; padding:0; }
0으로 초기화해줌. 선택자를 *로 써도 됨 (*=select all).
